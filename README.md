# Caixa Eletrônico em Java

Projeto desenvolvido para praticar lógica de programação e Programação Orientada a Objetos (POO) em Java.

## Funcionalidades

* Consultar saldo
* Realizar depósitos
* Realizar saques
* Fazer transferências
* Visualizar última operação
* Consultar extrato

## Conceitos utilizados

* Classes e Objetos
* Métodos
* Scanner
* ArrayList
* Estruturas de decisão (if/switch)
* Estruturas de repetição (while)
* Programação Orientada a Objetos

## Tecnologias

* Java
